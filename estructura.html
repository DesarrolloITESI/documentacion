Descarga directa de la colección

Flujo de Navegación

  1. Login: Usuario ingresa credenciales de la API (api_client:SecurePass2024!)
  2. Validación: Se hace petición al endpoint /auth/token para verificar
  3. Menú Principal: Si es válido, se muestra el menú con las opciones
  4. Navegación: Usuario puede acceder a ejercicio, documentación o descargas
  5. Autenticación persistente: Todas las páginas verifican el token almacenado
Nota: La autenticación se mantiene durante la sesión usando sessionStorage. Si el token expira o se cierra el navegador, el usuario debe volver a hacer login.
