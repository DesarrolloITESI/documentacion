Ejercicio Técnico Fullstack

Desarrolla una aplicación web que consuma nuestra API de órdenes

Información General
  • Empresa: Easy-Shop (E-commerce Analytics Platform)
  • Posición: Desarrollador Fullstack
  • Duración: 2-3 sesiones de 2-3 horas cada una
  • Modalidad: Live Coding - Evaluación en tiempo real

Objetivo Principal

Desarrollar una aplicación web básica que consuma nuestra API de órdenes y muestre los datos de manera simple.
La evaluación se centra en el proceso de desarrollo, no en el resultado final.
Backend (Sesión 1)
1. Consumir API de E-commerce
  • Obtener token de autenticación usando credenciales básicas
  • Consumir endpoint de órdenes del marketplace
  • Implementar paginación básica
2. Crear Base de Datos Relacional
  • Recomendado: PostgreSQL
  • También aceptamos: MySQL, SQLite
  • Diseñar esquema para almacenar órdenes
Frontend (Sesión 2)
3. Construir Backend API
  • Recomendado: Node.js con Express
  • También aceptamos: Python, PHP, Java
  • Endpoints para servir datos al frontend
4. Desarrollar Frontend
  • Recomendado: Angular
  • También aceptamos: React, Vue, vanilla JS
  • Dashboard con visualización de órdenes
Dashboard Requerido (Sesión 3)
Funcionalidades Principales:
  • Tabla con listado de órdenes
  • Modal con detalles de cada orden
  • Filtros por plataforma (Amazon, eBay, Etsy)
  • Búsqueda por email o ID de orden
Gráficos Requeridos:
  • Gráficos de barras horizontales
  • Filtrado por día específico
  • Agrupado por plataforma de venta
  • Mostrar totales de ventas por plataforma
